# Tokenizer robustness — dev-v1.1, bm25

The same nominal grid (chunk sizes, budgets B ∈ [200, 400, 800, 1600]) measured in two token units: regex word, cl100k_base BPE. Chunk sizes, budgets, and metric token sets all switch units together, and both grids score the identical 2,400-question sample. Paired bootstrap deltas (10,000 resamples) are computed within each unit; a claim is unit-robust when its sign and significance agree across the unit columns.

Generated by `python -m experiments.summarize_tokenizers` from `results/raw/` — do not edit by hand.
